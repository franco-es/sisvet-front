
# SIS-VET


SIS-VET Front-End, aplicaciòn SPA creada en React, utilizando axios como metodo de llamadas HTTP.


## Run Locally

Clone the project

```bash
  git clone https://github.com/franco-es/sisvet-front.git
```

Go to the project directory

```bash
  cd sisvet_front
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
